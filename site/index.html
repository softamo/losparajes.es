Apartamentos Rurales Los Parajes es un conjunto de ocho casas, situadas en Fuentelencina, Guadalajara.
Cada una de las casas toma el nombre de una zona del pueblo:
El Prado, El Parral, El Val,
El Palomar, El Molino, El San Agustin,
El Tallar, El Rehollo.

En este portal Web le mostramos nuestros servicios, le sugerimos

actividades y rutas turísticas de interes, le recomendamos que consulten nuestras Tarifas y se pongan en contacto con nosotros para cualquier duda o para realizar una reserva.
